# 🛒 Ecommerce Sales Analysis (Python + SQL + Visualization)

## 📌 Problem Statement

Ecommerce businesses generate large volumes of transactional data. This project analyzes sales data to uncover revenue trends, customer behavior, and product performance using Python and SQL.

---

## 📁 Project Overview

This project combines **SQL + Python** to perform end-to-end data analysis:

* SQL queries executed using Python cursor
* Data processing using Pandas
* Visualization using Matplotlib and Seaborn

---

## 📊 Key Analysis Performed

* Total Revenue Analysis
* Monthly Sales Trends
* Top-Selling Products
* Customer Purchase Behavior
* Category-wise Performance
* Profit / Sales Distribution

## 🔍 Key Insights

* A small number of products contribute to a large portion of total revenue (Pareto principle)
* Sales show clear trends over time, indicating seasonality
* Certain categories consistently outperform others in terms of revenue
* Customer purchasing patterns reveal repeat buying behavior
